# PRIVACY POLICY

**Gate Barcode Scanner System**

Effective Date: March 2026

---

## 1. Introduction

This Privacy Policy describes how the Gate Barcode Scanner System ("the System") collects, uses, processes, and protects information in connection with its operation.

The System is a locally hosted barcode scanning and access verification application designed to verify authorization status of scanned barcodes.

By operating or accessing the System, users acknowledge and agree to the practices described in this Privacy Policy.

---

## 2. Information Collected and Processed

The System processes the following categories of information:

### 2.1 Barcode Data

* Barcode values scanned through a connected camera device
* Authorization status (authorized or unauthorized)
* Timestamp of each scan event

This information is used solely for access verification and monitoring purposes.

### 2.2 Registered Authorization Codes

Authorized barcode values are stored locally in a JSON configuration file maintained by the system administrator.

### 2.3 Scan History

Recent scan results (including barcode value, timestamp, and authorization status) may be temporarily stored in system memory for real-time display and administrative monitoring.

### 2.4 Camera Data

When the scanner interface is used:

* The device camera captures live image frames.
* Captured frames are transmitted to the local server for barcode decoding.
* Image data is processed in volatile memory.
* Image files are not permanently stored unless explicitly modified by the system operator.

---

## 3. Purpose of Processing

Information processed by the System is used exclusively for:

* Access control verification
* Operational monitoring
* Administrative review of scan activity
* System functionality and debugging (if logging is enabled)

The System does not use collected information for marketing, advertising, profiling, or analytics purposes.

---

## 4. Data Storage and Retention

### 4.1 Local Storage

All data is stored locally on the server where the System is installed, unless the operator configures otherwise.

### 4.2 Retention Period

* Registered codes remain stored until manually removed by the administrator.
* Scan history is limited in size and automatically overwritten as new scan events occur.
* No image files are retained under default configuration.

---

## 5. Data Sharing and Disclosure

The System does not:

* Sell personal information
* Share information with third parties
* Transmit data to external cloud services
* Integrate third-party analytics tools

Information may only be accessed by authorized administrators with access to the hosting server.

---

## 6. Security Measures

Reasonable administrative and technical measures should be implemented by the system operator to protect stored information, including but not limited to:

* Restricting server access
* Securing network connections
* Protecting configuration files
* Maintaining firewall and operating system security

The System’s security level depends on the hosting environment configured by the operator.

---

## 7. Operator Responsibility

The individual or organization deploying the System is solely responsible for:

* Compliance with applicable data protection laws
* Informing users that barcode scanning is taking place
* Securing the server and network infrastructure
* Managing registered codes appropriately

---

## 8. International Use

If the System is made accessible over the internet or used in multiple jurisdictions, the operator must ensure compliance with applicable local and international data protection regulations.

This Privacy Policy may be updated to reflect changes in system functionality, legal requirements, or operational practices. The updated version shall supersede prior versions upon publication.
